Exact Winding Tables

Date: 2026-01-13

Air-core solenoid (Wheeler, inches): L(uH) = (r^2 * N^2) / (9r + 10l).

Select coil form diameter, length, and wire spacing to compute r and l.

For each turn count N, compute L and build a turns vs L table.

Ferrite core: L(nH) = AL * N^2; N = sqrt(L(nH) / AL).

Measure AL if possible for accurate tables.
