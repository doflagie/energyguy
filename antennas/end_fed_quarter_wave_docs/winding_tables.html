Calculating Winding Tables

Date: 2026-01-12

  • Winding tables are used for loading coils or matching transformers.
  • Select target inductance L and coil form dimensions.
  • Air-core solenoid: L(uH) = (r^2 * N^2) / (9r + 10l).
  • Ferrite cores: L(nH) = AL * N^2.
  • Create a table of turns vs inductance for tap points.
